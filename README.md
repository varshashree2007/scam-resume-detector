# 🛡 AI Scam & Resume Detector

An AI-powered web application that detects scam messages and analyzes resumes using machine learning and NLP techniques.

---

## 🚀 Features

- 📩 Spam Message Detection using ML model
- 📄 Resume Analyzer with scoring system
- 🎯 Detects suspicious patterns and fake claims
- 🌐 Live Web Application

---

## 🧠 Tech Stack

- Python
- Scikit-learn
- Streamlit
- NLP (TF-IDF)

---

## ⚙ How It Works

### 🔹 Scam Detection
- Uses TF-IDF Vectorization
- Trained with Naive Bayes model
- Predicts spam or safe messages

### 🔹 Resume Analyzer
- Extracts text from PDF
- Checks for:
  - Skills
  - Suspicious phrases
  - Structure
- Generates score and flags

---

## 📊 Model Accuracy

Achieved ~95% accuracy using trained dataset.
